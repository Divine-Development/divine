Divine Commands

servers

Description: Owner only command!

Usage: !servers

viewsettings

Description: Get the guild's settings! (Administrator permission required)

Usage: !viewsettings

removevip

Description: Remove a VIP member from the Database. (Owner only)

Usage: !removevip [user]

data

Description: Get my ping!

Usage: !data

setup

Description: Setup the bot for your server. (Administrator permissions required or Admin role after being set-up via the setup command!)

Usage: !setup [system] [value]

create

Description: Add the configuration to a guild (Owner only)

Usage: !create

reload

Description: Force reload staff data or guild settings. (Owner only)

Usage: !reload

suggest

Description: Create a suggestion inside a server with me! (Must be setup via !setup [suggestions] [channel])

Usage: !suggest

checkupdate

Description: Check the github repository for updates! (Owner Only)

Usage: !checkupdate

ping

Description: Check my ping!

Usage: !ping

addvip

Description: Add a VIP member to the Database. (Owner only)

Usage: !addvip [user]

addstaff

Description: Add a staff member to the Database. (Owner only)

Usage: !addstaff [user]

help

Description: Get help with using the bot!

Usage: !help

removestaff

Description: Remove a staff member from the Database. (Owner only)

Usage: !removestaff [user]
